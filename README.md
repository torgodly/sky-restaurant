# موقع مطعم سكاي — Sky Restaurant

موقع ثابت (Static) بصفحة واحدة — HTML + Tailwind CSS (CDN) + JavaScript. عربي RTL بالكامل، مصمم **Mobile-First**.

## التشغيل

افتح `index.html` مباشرة في المتصفح، أو شغّل سيرفر محلي:

```bash
npx serve .
```

## الصور

الصور محمّلة من حساب الانستغرام [@sky_eatly](https://www.instagram.com/sky_eatly/) وموجودة في:

- `images/` — الصور الرئيسية (`hero-burger.jpg`، `fries-cup.jpg`، `family-box.jpg`، `combo-flatlay.jpg`، `storefront.jpg`، `logo.jpg`) — صور دعائية من الانستغرام
- `images/gallery/` — صور قسم «من أجواء سكاي» — صور دعائية من الانستغرام
- `images/menu/` — **صور الأصناف الحقيقية** (من مجلد sky item): `beef-combo` · `chicken-combo` · `fries` · `sauce` · `soda` · `sundae` · `family-box` · `mega-box`
- `images/ig/` — النسخ الأصلية المحمّلة من الانستغرام (13 صورة) — احتياطي

لاستبدال أي صورة: ضع الصورة الجديدة بنفس الاسم والمسار.

## تعديل المنيو

كل بيانات المنيو في أعلى ملف [js/main.js](js/main.js) داخل المصفوفة `MENU` — عدّل الأسماء والأوصاف والصور من هناك مباشرة. **المنيو بلا أسعار** — التصميم على شكل منيو مطبوع: الصنف الذي له `img` يظهر ككرت بصورة، والصنف بدون `img` يظهر كسطر نصي أنيق.
